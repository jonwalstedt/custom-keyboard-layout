# Custom keyboard layout

- Use [ukelele](https://www.macupdate.com/app/mac/14495/ukelele) to remap keys an create a `.bundle` file
- Symlink `ln -s osx/U.S-dev-v0.0.1.bundle /Library/Keyboard Layouts/`
